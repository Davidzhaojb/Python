# 安装
```
1.npm install hexo-cli -g (全局安装hexo)

2.hexo init <文件夹名称>

3.cd <文件夹名称>

4.npm install(下载依赖，有时候会有依赖没办法正常下载的问题，需要进入nood_modules文件夹重新下载一次)

5.hexo server (或者 hexo s) 运行网站
```
#### 注：安装hexo的前提是要安装nodejs跟git，方法可自行百度
